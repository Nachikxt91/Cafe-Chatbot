# Cafe Chatbot Backend

AI-powered cafe assistant using LangChain, Groq LLM, and RAG with Pinecone vector database.

## Features
- Multi-agent conversational system
- Menu recommendations
- Order management
- RAG-based information retrieval

## API Documentation
Visit `/docs` for interactive API documentation.

## Tech Stack
- FastAPI
- LangChain
- Groq API
- Pinecone Vector DB
- Sentence Transformers
